Kunes | TASS TechAI - TASS_TechAI_v1_Final

This is a Vite + React mobile-first project ready for deployment on Vercel.

Run locally:
1. npm install
2. npm run dev

Deploy to Vercel:
1. Push to a GitHub repo
2. Import in Vercel (Build: npm run build, Output dir: dist)

Notes:
- Chat is a local demo. To integrate OpenAI, add a secure server endpoint.
- Logo placeholder included at /src/assets/logo.png
